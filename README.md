# CS115: Principles of Functional Programming

This is the GitHub Codespaces template for **CS115: Principles of Functional Programming**. We have prepared a full Haskell environment (GHC, HLS, Ormolu) ready for use directly in your browser.

## Quick Start

To ensure your work is safely backed up to your personal GitHub account, follow these steps:

1.  **Open Terminal** (Press `` Ctrl + ` ``) and type:
    ```bash
    ./.devcontainer/setup.sh
    ```
2.  **Authenticate with GitHub**: Follow the steps in the Terminal:
    - Select **Yes** to generate a new SSH key.
    - Press **Enter** to skip the Passphrase and filename steps.
    - **Copy the 8-digit code** shown in the Terminal.
    - Press **Enter** to open the login page, then paste the code to **Authorize**.
    - Once finished, the system will automatically create a repository named `26cs115haskell` in your account.

## How we work

-   **`homework/` folder**: All your code and assignments should be saved in this folder (organized by week: w01, w02, ...).
-   **Autosync System**: Don't worry about forgetting to save. The system will automatically backup (Commit & Push) your work in the `homework/` folder to GitHub every 10-30 minutes if the workspace is idle.
-   **Ready-to-use Tools**: We have pre-installed GHC, Haskell Language Server (HLS), and the Ormolu code formatter for you.
