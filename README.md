# CS115 Codespace

This repository is meant to be used with GitHub Codespaces.

## First time setup — save your work to GitHub

Once your Codespace is open, run this once in the terminal:

```bash
bash .devcontainer/setup.sh
```

This will create a private repo called `26cs115repo` on your GitHub and push your work there.
